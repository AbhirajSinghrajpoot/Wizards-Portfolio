## 🚀 Deployment

This portfolio utilizes continuous deployment (CI/CD) via GitHub Actions.

### Deployment Flow
1. Code changes are pushed to the `main` branch.
2. The GitHub Actions workflow (`deploy.yml`) automatically triggers.
3. The site is built and pushed to GitHub Pages.
4. Concurrently deployed and hosted on Vercel for optimal performance.

## 💻 Local Development Setup

Want to run this project on your local machine? Follow these steps:

**1. Clone the repository:**
```bash
git clone [https://github.com/AbhirajSinghrajpoot/Wizards-Portfolio.git](https://github.com/AbhirajSinghrajpoot/Wizards-Portfolio.git)
cd Wizards-Portfolio
```

**2. Launch the application:**
* Simply double-click `index.html` to open it in your browser.
* *Alternatively (Recommended)*, use a local development server like VS Code's "Live Server" extension or Python's built-in server:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
